# data/samples/

Reserved for non-confidential, illustrative or synthetic data used to demonstrate the
SignalScope AI workflow — for example, in documentation or a future portfolio
walkthrough. Unlike `data/input/` and `data/output/`, files placed here are intended
to be committed to version control, so nothing containing real client or brand data
should ever be placed in this folder.

No sample data has been created yet. See [CURRENT_SPRINT.md](../../CURRENT_SPRINT.md)
for current project status.
